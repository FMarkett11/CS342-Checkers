# CS342 Checkers

A Java-based Checkers game created for a CS342 course project featuring classic gameplay mechanics, turn-based logic, and king promotion.

---

## Features

- Classic checkers gameplay (No forced jump)
- Turn-based player system
- King promotion mechanics
- Move validation via highlights & repeat move upon capture
- Simple interactive game interface

---

## Tech Stack

- **Language:** Java
- **Concepts:** Object-Oriented Programming, Multithreaded programming, Server/Client Connectivity
- **Tools:** Git/GitHub, IntelliJ IDEA, Maven, JavaFX
